## **Módulo 1: Introducción a Arduino**
1. **Clase 1: Introducción y configuración inicial**
   - ¿Qué es Arduino?
   - Diferentes modelos de placas Arduino
   - Instalación del Arduino IDE
   - Primera conexión: Verificación de la instalación

2. **Clase 2: Componentes básicos**
   - Descripción de los componentes incluidos en el kit (resistencias, LEDs, botones, potenciómetros, sensores, etc.)
   - Uso de una protoboard

## **Módulo 2: Proyectos Básicos**
3. **Clase 3: Encender un LED**
   - Conceptos de electricidad básica (corriente, voltaje, resistencia)
   - Primer programa: Encender y apagar un LED (Blink)
   - Uso de resistencias

4. **Clase 4: Botón y LED**
   - Introducción a las entradas digitales
   - Control de un LED con un botón

5. **Clase 5: PWM y control de brillo**
   - Introducción al PWM (Pulse Width Modulation)
   - Control de brillo de un LED usando PWM

## **Módulo 3: Sensores y Actuadores**
6. **Clase 6: Sensor de temperatura y humedad**
   - Conexión y programación del sensor DHT11/DHT22
   - Mostrar datos en el monitor serial

7. **Clase 7: Pantalla LCD**
   - Conexión y configuración de una pantalla LCD
   - Mostrar la temperatura y humedad en la pantalla LCD

8. **Clase 8: Control de servomotor**
   - Introducción a los servomotores
   - Conexión y programación básica para mover un servomotor

9. **Clase 9: Sensor de luz (LDR)**
   - Conexión y uso de un sensor de luz (fotocélula o LDR)
   - Proyecto: Controlar el brillo de un LED basado en la luz ambiental

## **Módulo 4: Proyectos Intermedios**
10. **Clase 10: Sensor ultrasónico**
    - Conexión y uso del sensor ultrasónico HC-SR04
    - Proyecto: Medir distancias y mostrarlas en el monitor serial

11. **Clase 11: Control de motores DC**
    - Introducción a los motores DC
    - Uso de un driver de motor (L298N)
    - Proyecto: Control de velocidad y dirección de un motor DC

12. **Clase 12: Control de un servomotor con potenciómetro**
    - Conexión y programación
    - Proyecto: Mover un servomotor en base a la posición de un potenciómetro

## **Módulo 5: Proyectos Avanzados**
13. **Clase 13: Robot seguidor de línea**
    - Sensores de infrarrojos
    - Conceptos de robótica básica
    - Programación y montaje del robot seguidor de línea

14. **Clase 14: Conexión a Internet (IoT)**
    - Uso del módulo Wi-Fi ESP8266
    - Enviar datos a una plataforma IoT (Ej. ThingSpeak)
