# Cómo Contribuir

¡Tu ayuda es valiosa! Sigue estos pasos simples para contribuir a mejorar nuestra documentación:

## 1. Crea un Fork del Repositorio

1. Visita nuestro [repositorio en GitHub](https://github.com/bysmax/bysmax-blog)
2. Haz clic en el botón "Fork" en la esquina superior derecha
3. Esto creará una copia del repositorio en tu cuenta de GitHub

## 2. Realiza tus Cambios

1. Clona tu fork a tu computadora local
2. Crea una nueva rama para tus cambios
3. Realiza las modificaciones o adiciones necesarias
4. Asegúrate de seguir el estilo y formato existente

## 3. Envía un Pull Request

1. Sube tus cambios a tu fork
2. Ve a la página principal del repositorio original
3. Haz clic en "Pull Request"
4. Selecciona tu rama y describe los cambios que realizaste
5. Envía el Pull Request

## 4. Espera la Revisión

- Tu Pull Request será revisado por los mantenedores
- Podrían sugerir algunos cambios o mejoras
- Una vez aprobado, tus cambios se fusionarán con el repositorio principal

## ¡Con tu ayuda mejoraremos este contenido!

Cada contribución cuenta. Ya sea corrigiendo errores tipográficos, mejorando explicaciones o agregando nueva documentación, tu ayuda hace que este recurso sea mejor para todos.

¿Tienes preguntas? No dudes en abrir un issue en el repositorio para discutir tus ideas o preguntar dudas.

¡Gracias por contribuir! 🚀
