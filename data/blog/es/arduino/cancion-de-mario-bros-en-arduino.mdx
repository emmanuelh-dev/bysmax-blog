## Materiales Necesarios para Reproducir la Canción de Mario Bros en Arduino

Para este proyecto de Arduino, necesitarás los siguientes materiales:

- **Arduino Uno**: Una placa Arduino es esencial para este proyecto.
- **Buzzer Piezoeléctrico**: Este componente es responsable de producir el sonido.
- **Cables de Conexión**: Para conectar el buzzer al Arduino.
- **Protoboard**: Facilita la conexión de los componentes.
- **Simulación en tinkercad**: [Puedes ver, copiar o editar la simulación de tinkercad.](https://www.tinkercad.com/things/2fBckSQKaTs-arduino-mario-bros-theme?sharecode=t-9oZ6SSg7_Q5otvH1vkC13bRsIT7VlRffsTkFo8LMw)

![Conexión del Buzzer](/static/images/arduino/buzzeruno.jpg)

## Paso a Paso: Conectando el Buzzer al Arduino

1. **Conectar el Buzzer**:
   - Conecta el terminal positivo del buzzer al pin digital 3 del Arduino.
   - Conecta el terminal negativo del buzzer a un pin de tierra (GND) en la placa Arduino.
2. **Conexiones Adicionales**:
   - Usa cables de conexión para asegurar que las conexiones sean firmes y seguras.
   - Coloca el buzzer en una protoboard para facilitar la conexión y evitar cortocircuitos.

}


void buzz(int targetPin, long frequency, long length) {

  digitalWrite(13, HIGH);

  long delayValue = 1000000 / frequency / 2;

  long numCycles = frequency * length / 1000;

  for (long i = 0; i < numCycles; i++) {

    digitalWrite(targetPin, HIGH);

    delayMicroseconds(delayValue);

    digitalWrite(targetPin, LOW);

    delayMicroseconds(delayValue);

  }

  digitalWrite(13, LOW);


}

## Explicación Técnica: Cómo Funciona el Código

El código utiliza una biblioteca de Arduino para definir las notas musicales y sus duraciones. Cada nota se reproduce usando la función `tone()`, que envía una señal al buzzer conectado al pin digital 3 del Arduino. La duración de cada nota se controla con la variable `noteDuration`, y hay pausas entre las notas para que la melodía sea reconocible. Si te interesa profundizar un poco mas, puedes revisar la pagina de arduino, en la que se explica esta funcion https://docs.arduino.cc/built-in-examples/digital/toneMelody/
