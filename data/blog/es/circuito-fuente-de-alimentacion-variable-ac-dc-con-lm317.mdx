Crear una **fuente de alimentación variable AC-DC** es fundamental para cualquier laboratorio de electrónica o taller de reparaciones. En este artículo completo, te enseñaremos cómo construir una fuente de alimentación ajustable de 3 amperios utilizando el popular regulador de voltaje [LM317](https://electronica.bysmax.com/componentes-electronicos/lm317t). Este componente versátil permite obtener voltajes desde 1.25V hasta 37V, convirtiéndolo en la opción perfecta para alimentar una amplia variedad de proyectos electrónicos.

## ¿Qué es el LM317 y por qué es ideal para fuentes variables?

El [LM317](https://electronica.bysmax.com/componentes-electronicos/lm317t) es un **regulador de voltaje ajustable lineal** que se ha convertido en el estándar de la industria para aplicaciones de voltaje variable. A diferencia de los reguladores fijos como el [LM7812](https://electronica.bysmax.com/componentes-electronicos/lm7812) o [LM7805](https://electronica.bysmax.com/componentes-electronicos/lm7805), el LM317 permite ajustar el voltaje de salida mediante resistencias externas o un potenciómetro.

## Lista Completa de Materiales para Fuente Variable con LM317

Para construir una **fuente de alimentación variable de 3A** con LM317, necesitarás los siguientes componentes electrónicos:

### Componentes Principales
- **[LM317T](https://electronica.bysmax.com/componentes-electronicos/lm317t)**: Regulador de voltaje ajustable (encapsulado TO-220)
- **Transformador 18V-20V / 3A**: Para reducir el voltaje de red a nivel manejable
- **[Puente rectificador](https://electronica.bysmax.com/componentes-electronicos/puente-diodos)**: 4 diodos 1N5408 o puente integrado de 4A
- **[LM7812](https://electronica.bysmax.com/componentes-electronicos/lm7812)** (opcional): Para pre-regulación y mejor estabilidad

### Capacitores para Filtrado
- **Capacitor electrolítico 4700µF/35V**: Filtrado principal después del rectificador
- **Capacitor electrolítico 470µF/25V**: Filtrado en la entrada del LM317
- **Capacitor cerámico 100nF**: Desacoplo en entrada del LM317
- **Capacitor electrolítico 10µF/25V**: Estabilización en la salida
- **Capacitor cerámico 10nF**: Filtrado de alta frecuencia en salida

### Componentes de Control y Protección
- **Potenciómetro lineal 5KΩ**: Para ajuste del voltaje de salida
- **Resistencia 240Ω (1/4W)**: Resistencia fija para el LM317
- **Resistencia 1KΩ**: Para LED indicador
- **LED verde**: Indicador de funcionamiento
- **Fusible 3A**: Protección del circuito primario
- **Disipador de calor**: Para el LM317 (mínimo 5°C/W)

### Instrumentación (Opcional)
- **Voltímetro digital**: Para monitorear voltaje de salida
- **Amperímetro**: Para control de corriente
- **Switch ON/OFF**: Interruptor principal

## Montaje Paso a Paso: Construcción de Fuente Variable con LM317

### Paso 1: Diseño del Transformador y Rectificación

**Importante**: Antes de trabajar con voltajes de red, asegúrate de tomar todas las [medidas de seguridad en electrónica](https://electronica.bysmax.com/tutoriales/seguridad-electronica).

1. Conecta el **transformador** (18V-20V / 3A) a la red de 110V-120V
2. Conecta la salida del transformador al **[puente rectificador](https://electronica.bysmax.com/componentes-electronicos/puente-diodos)**
3. Esto convertirá la corriente AC en DC pulsante

![Preparar la Fuente de Alimentación Primaria](/static/images/transformador.png)

### Paso 2: Filtrado Principal y Estabilización

1. Conecta el **capacitor de 4700µF** inmediatamente después del puente rectificador
2. **¡Atención a la polaridad!** El positivo va al ánodo de los diodos
3. Añade el **capacitor de 100nF** en paralelo para filtrar ruido de alta frecuencia
4. En este punto deberías obtener aproximadamente 25V DC

### Paso 3: Configuración del LM317 - El Corazón del Circuito

El [LM317](https://electronica.bysmax.com/componentes-electronicos/lm317t) tiene **3 terminales principales**:

- **Pin 1 (ADJ)**: Terminal de ajuste
- **Pin 2 (OUT)**: Salida regulada
- **Pin 3 (IN)**: Entrada de voltaje

**Conexiones del LM317:**
1. **Entrada (Pin 3)**: Conecta al voltaje DC filtrado (25V aproximadamente)
2. **Salida (Pin 2)**: Conecta a la carga y al circuito de realimentación
3. **Ajuste (Pin 1)**: Conecta al potenciómetro de 5KΩ

**Fórmula del voltaje de salida:**
```
Vout = 1.25V × (1 + R2/R1)
Donde R1 = 240Ω (fija) y R2 = potenciómetro variable
```

![Circuito Fuente de Alimentación Variable AC-DC con LM317](/static/images/fuente-regulada-lm7812.png)

### Paso 4: Circuito de Ajuste y Estabilización

1. Conecta la **resistencia de 240Ω** entre la salida (Pin 2) y ajuste (Pin 1) del LM317
2. Conecta el **potenciómetro de 5KΩ** entre el terminal de ajuste (Pin 1) y tierra
3. Añade el **capacitor de 10µF** entre salida y tierra para mejor respuesta transitoria
4. Conecta el **capacitor de 10nF** en paralelo para filtrar ruido

### Paso 5: Protección Térmica y Montaje Final

1. **Instala el disipador de calor** en el LM317 usando pasta térmica
2. **Cálculo de disipación**: P = (Vin - Vout) × Iout
3. Para 3A a diferencia de 10V: P = 30W (¡Requiere disipador grande!)
4. Conecta el **LED indicador** con su resistencia de 1KΩ
5. Instala el **fusible de 3A** en el circuito primario

### Paso 6: Pruebas de Funcionamiento y Calibración

**Lista de verificación antes del primer encendido:**

1. ✅ **Polaridad de capacitores electrolíticos**
2. ✅ **Conexiones del LM317** (IN, OUT, ADJ)
3. ✅ **Disipador térmico** correctamente instalado
4. ✅ **Fusibles** en su lugar
5. ✅ **No hay cortocircuitos** entre pistas

**Procedimiento de prueba:**
1. Conecta un **[multímetro](https://electronica.bysmax.com/herramientas/multimetro)** en la salida
2. Enciende la fuente sin carga
3. Gira el potenciómetro y verifica que el voltaje varíe entre **1.25V y 30V**
4. Prueba con cargas progresivas: 100mA, 500mA, 1A, 2A, 3A
5. Monitorea la temperatura del LM317 durante las pruebas

## Características Técnicas del LM317 - Especificaciones Completas

El [LM317](https://electronica.bysmax.com/componentes-electronicos/lm317t) es un **regulador lineal ajustable** con las siguientes especificaciones:

### Parámetros Eléctricos
- **Rango de voltaje de salida**: 1.25V a 37V
- **Corriente máxima de salida**: 1.5A (con disipador adecuado)
- **Voltaje de entrada máximo**: 40V
- **Dropout voltage**: 2V típico
- **Regulación de línea**: 0.01% típica
- **Regulación de carga**: 0.1% típica
- **Corriente de ajuste**: 50µA típica
- **Ruido de salida**: 0.003% del Vout

### Protecciones Integradas
- **Protección térmica**: Limitación a 125°C
- **Protección contra sobrecorriente**: Plegado de corriente
- **Protección SOA**: Área de operación segura

### Ventajas vs Reguladores Fijos
Comparado con reguladores fijos como [LM7805](https://electronica.bysmax.com/componentes-electronicos/lm7805) o [LM7812](https://electronica.bysmax.com/componentes-electronicos/lm7812):
- ✅ **Voltaje ajustable** vs voltaje fijo
- ✅ **Mayor flexibilidad** en aplicaciones
- ✅ **Mismas protecciones** integradas
- ❌ **Circuito más complejo** (requiere componentes externos)

## Análisis de Ventajas y Consideraciones del LM317

### ✅ Principales Beneficios

**Flexibilidad Operacional**
- **Voltaje ajustable**: Desde 1.25V hasta 37V con un solo componente
- **Precisión alta**: Tolerancia típica de ±1% en la regulación
- **Respuesta rápida**: Excelente regulación ante cambios de carga
- **Compatibilidad universal**: Funciona con cualquier voltaje de entrada adecuado

**Facilidad de Implementación**
- **Circuito simple**: Requiere pocos componentes externos
- **Documentación extensa**: Décadas de uso en la industria
- **Disponibilidad**: Fácil de conseguir y económico
- **Protecciones integradas**: Térmica y contra sobrecorriente incluidas

**Calidad de Salida**
- **Bajo ruido**: Ideal para [circuitos analógicos](https://electronica.bysmax.com/tutoriales/circuitos-analogicos)
- **Estabilidad**: Excelente regulación de línea y carga
- **Ripple bajo**: Mínimas ondulaciones en la salida

### ⚠️ Consideraciones Importantes

**Gestión Térmica**
- **Disipación de calor**: P = (Vin - Vout) × Iout puede ser considerable
- **Eficiencia limitada**: Reguladores lineales típicamente 60-70%
- **Disipador obligatorio**: Para corrientes superiores a 500mA
- **Ubicación**: Requiere ventilación adecuada

**Limitaciones de Diseño**
- **Dropout voltage**: Mínimo 2V entre entrada y salida
- **Corriente máxima**: 1.5A sin modificaciones (requiere boost para 3A)
- **Voltaje mínimo**: No puede bajar de 1.25V
- **Consumo en standby**: Corriente quiescente de aproximadamente 10mA

**Alternativas Modernas**
Para aplicaciones que requieren mayor eficiencia, considera:
- **Reguladores conmutados**: Como el [LM2596](https://electronica.bysmax.com/componentes-electronicos/lm2596)
- **Módulos DC-DC**: Para prototipos rápidos
- **Fuentes digitales**: Con control por microcontrolador

## Consejos Profesionales para Construcción Exitosa

### 🔧 Mejores Prácticas de Montaje

**1. Cálculo Térmico Previo**
```
Potencia disipada = (Vin - Vout) × Iout
Ejemplo: (25V - 5V) × 2A = 40W
Disipador requerido: Rth < 2°C/W
```

**2. Layout del PCB (si usas PCB personalizado)**
- **Planos de tierra amplios**: Mejor disipación térmica
- **Pistas gruesas**: Para corrientes altas (mínimo 1mm por ampere)
- **Separación térmica**: Aisla el LM317 de componentes sensibles
- **Vías de disipación**: Conecta el tab del LM317 al plano de tierra

**3. Selección de Componentes Críticos**
- **Transformador**: Dimensiona 20% por encima de la potencia nominal
- **Capacitores electrolíticos**: Usa tipos de baja ESR para mejor filtrado
- **Disipador**: Calcula considerando temperatura ambiente máxima
- **Potenciómetro**: Tipo lineal (no logarítmico) para ajuste proporcional

### 🛡️ Protecciones Adicionales Recomendadas

**Protección contra Sobretensión**
- **Varistor MOV**: En la entrada de AC
- **Diodo TVS**: En la salida DC para proteger cargas sensibles

**Protección contra Inversión de Polaridad**
- **Diodo schottky**: En serie con la salida (baja caída de voltaje)

**Limitación de Corriente Ajustable**
```
Agrega una resistencia de sense (0.1Ω) y un amplificador 
operacional para crear limitación de corriente programable
```

### 📊 Troubleshooting - Problemas Comunes

| Problema | Causa Probable | Solución |
|----------|----------------|----------|
| **Voltaje de salida bajo** | Dropout insuficiente | Aumentar Vin mínimo 2V |
| **Oscilación** | Capacitor de salida bajo | Aumentar C_out a 10µF |
| **Sobrecalentamiento** | Disipador insuficiente | Calcular y reemplazar disipador |
| **Regulación pobre** | Ripple alto en entrada | Aumentar capacitor de filtrado |
| **No ajusta voltaje** | Potenciómetro dañado | Verificar continuidad del pot |

### 🔍 Mediciones y Calibración

**Equipos necesarios:**
- **[Multímetro digital](https://electronica.bysmax.com/herramientas/multimetro)**: Precisión mínima 0.5%
- **Osciloscopio**: Para medir ripple (opcional)
- **Carga electrónica**: Para pruebas de regulación
- **Termómetro infrarrojo**: Para monitoreo térmico

**Procedimiento de calibración:**
1. **Sin carga**: Ajustar rango completo 1.25V-30V
2. **Con carga nominal**: Verificar regulación menor al 1%
3. **Respuesta transitoria**: Cambios súbitos de carga
4. **Medición de ripple**: Menor a 1mVpp para aplicaciones sensibles
