# ¡Bienvenidos al nuevo blog de bysmax!

Este blog es completamente de código abierto y está mantenido por bysmax. Si deseas agregar una nueva entrada, puedes hacerlo [aquí](https://github.com/emmanuelh-dev/bysmax-blog/new/main/data/blog). O editarla directamente.

Solo tienes que seguir la siguiente estructura:

En la sección del nombre de archivo, utiliza tu-nombre-de-ruta.mdx y automáticamente aparecerá:

```markdown
---
title: Tu título
date: '2020-12-21' // La fecha en la que lo creaste, cuanto más reciente, aparecerá antes
tags: ['next js', 'math', 'ols'] // Etiquetas para identificar tu publicación
draft: false // Si ya está publicado o no
summary: 'Cómo derivar el Estimador OLS con notación de matrices y un recorrido por la composición matemática utilizando markdown con la ayuda de KaTeX.' // Una descripción de lo que trata tu blog
---

Tu contenido aquí,
```

Puedes usar ChatGPT para darle formato de markdown. Si no te sientes seguro con tu escritura, aquí tienes una guía sobre lo que debes tener en cuenta: [Guía de Markdown de GitHub](https://emmanuelh.dev/blog/github-markdown-guide).
