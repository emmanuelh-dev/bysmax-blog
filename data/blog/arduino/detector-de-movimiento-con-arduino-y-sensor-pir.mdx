## Introducción al Proyecto: Detector de Movimiento con Arduino y Sensor PIR

Los sensores PIR (Infrarrojos Pasivos) son componentes económicos y versátiles utilizados en una variedad de dispositivos para detectar movimiento. Estos sensores son comunes en sistemas de seguridad, luces automáticas y muchos otros proyectos de automatización. En este artículo, exploraremos cómo construir un detector de movimiento con Arduino y un sensor PIR, específicamente el modelo HC-SR501. Aprenderás sobre los materiales necesarios, cómo realizar las conexiones, programar el Arduino y ajustar el sistema para obtener los mejores resultados.

## Materiales Necesarios

Para este proyecto, necesitarás los siguientes materiales:

- **Arduino UNO**: La plataforma de desarrollo principal para nuestro proyecto.
- **Sensor PIR HC-SR501**: El sensor de movimiento que utilizaremos.
- **Cables de conexión**: Para conectar el sensor al Arduino.
- **Protoboard**: Facilita las conexiones temporales sin necesidad de soldar.
- **Resistencias**: Según sea necesario para el montaje.
- **LED y resistencias asociadas**: Para indicar visualmente el movimiento detectado.
- **Fuente de alimentación**: Para energizar el Arduino.

### Conexión del Sensor PIR al Arduino

La conexión del sensor PIR al Arduino es bastante sencilla. El sensor HC-SR501 tiene tres pines:

1. **VCC**: Alimentación, se conecta al pin de 5V del Arduino.
2. **GND**: Tierra, se conecta al pin GND del Arduino.
3. **OUT**: Salida de señal, se conecta a uno de los pines digitales del Arduino, por ejemplo, el pin 2.

**Pasos para la conexión**:

1. Coloca el sensor PIR en la protoboard.
2. Conecta el pin VCC del sensor PIR al pin de 5V del Arduino usando un cable de conexión.
3. Conecta el pin GND del sensor PIR al pin GND del Arduino.
4. Conecta el pin OUT del sensor PIR al pin digital 2 del Arduino.

![Conexión del sensor PIR al Arduino](/static/images/arduino/arduino-pir-montaje.png)

### Programación del Arduino: Código Fuente

A continuación, te mostramos el código básico para hacer funcionar el sensor PIR con el Arduino. Este código hará que un LED se encienda cuando el sensor detecte movimiento.

```cpp
int pirPin = 2; // Pin donde se conecta el sensor PIR
int ledPin = 13; // Pin del LED integrado en el Arduino

void setup() {
  pinMode(pirPin, INPUT);
  pinMode(ledPin, OUTPUT);
  Serial.begin(9600); // Inicializa la comunicación serie
}

void loop() {
  int pirState = digitalRead(pirPin); // Lee el estado del sensor PIR
  if (pirState == HIGH) {
    digitalWrite(ledPin, HIGH); // Enciende el LED
    Serial.println("Movimiento detectado");
  } else {
    digitalWrite(ledPin, LOW); // Apaga el LED
    Serial.println("Sin movimiento");
  }
  delay(1000); // Espera 1 segundo
}
```

### Ajuste y Pruebas del Detector de Movimiento

Una vez que hayas cargado el código en tu Arduino, es hora de realizar algunos ajustes y pruebas para asegurarte de que todo funcione correctamente.

1. **Ajuste de Sensibilidad**: El sensor PIR HC-SR501 tiene dos potenciómetros que te permiten ajustar la sensibilidad y el tiempo de retardo. Gira estos potenciómetros hasta encontrar la configuración óptima para tu aplicación.
2. **Pruebas de Detección**: Coloca el sensor en diferentes ubicaciones y verifica su capacidad para detectar movimiento. Observa la salida en el monitor serie para asegurarte de que los cambios en el estado del sensor se registren correctamente.

### Aplicaciones Prácticas y Beneficios

Los sensores PIR tienen una amplia gama de aplicaciones tanto en entornos domésticos como industriales:

- **Automatización del hogar**: Los sensores PIR se utilizan para encender luces automáticamente cuando se detecta movimiento. Esto es especialmente útil en áreas como pasillos, baños y entradas. En mi experiencia personal, he visto estos sensores instalados en los baños de gimnasios, donde ayudan a ahorrar energía al encender las luces solo cuando hay alguien presente.
- **Seguridad**: Los sensores PIR son fundamentales en los sistemas de seguridad para detectar intrusos. Al integrar estos sensores con un sistema de alarma, se puede mejorar significativamente la seguridad de una vivienda o instalación.
- **Ahorro de energía**: En aplicaciones industriales, los sensores PIR se utilizan para reducir el consumo de energía. Por ejemplo, pueden apagar equipos cuando no se detecta movimiento en una zona determinada, lo que ayuda a reducir los costos operativos.
