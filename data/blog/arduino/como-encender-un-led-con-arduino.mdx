# Cómo Hacer Parpadear un led Con Arduino 1

En esta emocionante serie que hemos creado para acompañarte en tu trayectoria como entusiasta de la electrónica, nos sumergiremos en un proyecto fundamental y tremendamente útil.

Esta propuesta básica te permitirá dar tus primeros pasos en el fascinante universo de la electrónica.

## Materiales Necesarios

1. Arduino Uno
2. LED
3. Resistencia (generalmente de 220 ohmios)
4. Cables de conexión

## Conexiones

1. Conecta el ánodo del LED (el lado más largo) al pin digital 13 del Arduino Uno.
2. Conecta el cátodo del LED (el lado más corto) a través de la resistencia y luego a tierra (GND) en el Arduino.

## Programación

Utilizaremos el lenguaje de programación Arduino, basado en C/C++, para controlar el LED. Asegúrate de tener el entorno de desarrollo de Arduino instalado.

```arduino
// Definir el pin del LED
const int pinLed = 13;

void setup() {
  // Configurar el pin del LED como salida
  pinMode(pinLed, OUTPUT);
}

void loop() {
  // Encender el LED
  digitalWrite(pinLed, HIGH);

  // Esperar 1 segundo
  delay(1000);

  // Apagar el LED
  digitalWrite(pinLed, LOW);

  // Esperar 1 segundo
  delay(1000);
}

```

Este sencillo programa enciende y apaga el LED conectado al pin 13 con intervalos de un segundo. Puedes personalizar el código según tus necesidades.

## Carga del Programa

1. Conecta tu Arduino Uno a la computadora mediante un cable USB.
2. Abre el entorno de desarrollo Arduino.
3. Copia y pega el código en el editor.
4. Selecciona tu placa y puerto en las opciones del entorno.
5. Haz clic en "Subir" para cargar el programa en el Arduino.

¡Y listo! Ahora deberías ver cómo tu LED parpadea de acuerdo con el programa que has cargado. Este proyecto básico es solo el punto de partida, y puedes ampliar tus habilidades con Arduino explorando más funciones y sensores.

Puedes incluso añadir un botón que Agreue funcionamiento extra.
