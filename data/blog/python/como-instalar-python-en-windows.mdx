## Cómo Instalar Python en Windows

### Paso 1: Descarga de Python

Dirígete al sitio web oficial de Python en [python.org](https://www.python.org/downloads/), y sigue estos pasos:

1. Haz clic en "Descargas" en la barra de navegación.
2. Selecciona la última versión estable de Python para Windows.
3. Elige el instalador correspondiente a tu sistema operativo de 32 o 64 bits.

### Paso 2: Ejecución del Instalador

Una vez que descargues el instalador de Python, realiza lo siguiente:

1. Haz doble clic en el archivo descargado.
2. Marca la casilla "Agregar Python X.X a PATH" (donde "X.X" es la versión de Python).
3. Haz clic en "Instalar Ahora" y sigue las instrucciones en pantalla.

### Paso 3: Verificación de la Instalación

Después de la instalación, verifica si Python se instaló correctamente:

- Abre una ventana de comandos (cmd) o powershell.

- Ejecuta el siguiente comando: `python --version`.
![Python version](/static/images/pythonversion.png)
Deberías ver la versión de Python instalada.

### Paso 4: Instalación de un Editor de Código (Opcional)

Si lo prefieres, instala un editor de código para programar en Python de manera más cómoda. Algunas opciones populares son:

- [Visual Studio Code](https://code.visualstudio.com/) (Mi favorito)
- [PyCharm](https://www.jetbrains.com/pycharm/)
- [Atom](https://atom.io/)

Descarga e instala el editor de tu elección desde su sitio web oficial y comienza a programar en Python.

¡Listo! Ahora tienes Python instalado en Windows y estás listo para comenzar a programar. ¡Disfruta!
